Role: You are an expert ESL curriculum designer familiar with the Taiwan 108 Curriculum for Primary 5 and the TOEFL Primary Step 2 standards.

Task: Create a comprehensive English quiz and output it as a fully interactive, raw HTML web application.

Source Materials:

Attachment 1 (Core Content): The provided textbook unit/chapter material. First, analyze this content to identify the core vocabulary, grammar rules, sentence patterns, and themes. Quiz Requirements:

Target Audience: Primary 5 student in Taiwan.

CRITICAL FORMATTING RULE: Do NOT include any citation tags (e.g., "【1】") within the generated HTML quiz content, UI, or explanations.

Functionality:

Allow the student to select answers.

Upon submission, calculate the score.

Display a review section showing incorrect questions, the correct answers, and a clear, simple grammar explanation for why the answer is correct.
